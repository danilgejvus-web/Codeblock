# Codeblock

-каждый блок подгружает айдишник, название, булевые значения, входы/выходы и визуал<br>
-айдишники записываются в общий массив<br>
-массив перерабатывается в объекты, исполняющие друг друга и передающие друг другу данные<br>
-объекты исполняются и выполняется программа<br>
<br>
визуальные блоки:<br>
айди, имя, флаги функционала и визуала, вводимые в визуальной части значения<br>
<br>
исполнимые блоки:<br>
айди, метод, ссылка на последующий объект с передачей ему данных
