# Codeblock

-каждый блок подгружает айдишник, название, булевые значения, входы/выходы и визуал
-айдишники записываются в общий массив
-массив перерабатывается в объекты, исполняющие друг друга и передающие друг другу данные
-объекты исполняются и выполняется программа

визуальные блоки:
айди, имя, флаги функционала и визуала, вводимые в визуальной части значения

исполнимые блоки:
айди, метод, ссылка на последующий объект с передачей ему данных
